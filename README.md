# Dart Fundamentals

A collection of Dart fundamentals, examples, and exercises completed to build a solid foundation before learning Flutter.

## 📚 Topics Covered

* Variables and Data Types
* Operators and Expressions
* Conditional Structures
* Loops
* Functions
* Strings and Interpolation
* Null Safety

## 🎯 Goal

This repository documents my journey of learning the Dart programming language as preparation for Flutter development.

```text

├── loops.dart
├── functions.dart
├── strings_and_interpolation.dart
└── null_safety.dart
```

## 🚀 Getting Started

Clone the repository:

```bash
git clone <repository-url>
```

Run any example with the Dart SDK:

```bash
dart run filename.dart
```

Example:

```bash
dart run functions.dart
```

## 🛠 Requirements

* Dart SDK 3.x or later
